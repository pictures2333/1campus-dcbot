## 啟用方法

開啟``DCBOTSET.py``

```python
def TOKEN(): return ""
```

將 ``def TOKEN(): `` 的回傳值改成機器人的token

安裝以下模組
```
discord
requests
```
指令
```
pip3 install discord.py
pip3 install requests
```

開啟 ``bot.py``
